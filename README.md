# Read Documentation
### About Project
This is the Online Book reading platform in Django - python framework. There are 2 modules, user/reader and admin. Admin can manage whole website, including managing books on website and users request for new book. And user can read and add to save for read later and user can also make request a book which is not on website user can update profile, change password, reset password using email.
